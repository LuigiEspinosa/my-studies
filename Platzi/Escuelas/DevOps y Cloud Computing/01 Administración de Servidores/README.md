# Administración de Servidores

La administración de servidores engloba todas las actividades dirigidas a gestionar, monitorear y velar por la seguridad de los sistemas informáticos de una organización. El servidor es el cerebro que dirige y da vida a todas las operaciones. Este conjunto de ordenadores no solo almacena todos los archivos, sino que suministra la información precisa cuando es necesario.
