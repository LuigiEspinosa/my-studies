# Frontend con Vue.js

Vue.js es un primer gran framework frontend para aprender gracias a su suave curva de aprendizaje y fuerte presencia en la comunidad de JavaScript.

## ¿Qué es Vue.js?

Vue.js es una de las librerías de JavaScript más importantes para el desarrollo frontend. También lo conocemos como "el framework progresivo" porque puede ser implementado tanto como librería o como framework. Al usar un patrón 100% reactivo Vue permite actualizar el DOM en tiempo real al mismo tiempo que creamos interfaces basadas en componentes reutilizables.
