# HTML y CSS a Profundidad

## ¿Qué es HTML y CSS?

Nuestra primera responsabilidad como frontend developers es transformar diseños, prototipos interactivos y sistemas de diseño en código real con HTML y CSS.

HTML es el lenguaje para definir la estructura y contenido de cualquier sitio web en internet usando títulos, párrafos, enlaces, imágenes, videos, entre otros. CSS es el lenguaje para darle tamaño, color, ubicación e incluso animaciones a todos los elementos de nuestro HTML.
