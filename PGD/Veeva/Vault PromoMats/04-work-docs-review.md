# Working with Documents During the Review Process

## Workflows

Each document type in Vault PromoMats has a document lifecycle tied to it which can have one or more workflows associated with it.

When working with documents through the review and approval process, there will be times that you may need to start a workflow. Workflows can be started two ways, one way is via the Workflow Actions Menu, which displays if you have workflows available to you.

Once you select it you'll see available workflows, Send for Coordinate Review as an example on the right.

The second way is when you complete a task, a new workflow may be triggered. When a workflow is started, you'll be asked to populate information such as the due date and the reviewers/approvers.

### Emailing Workflow Participants

There may be times where you need to email the participants on a workflow to remind them of the task assigned to them. Vault PromoMats makes this easy to do from right within the tool.

- One way this can be accessed is from the 'Active Workflows' view on the Vault PromoMats home screen. Find the document/workflow you're looking for and open the 'Action Menu' and select 'Email Participants'.

- The second way is from the document iself, click the workflow icon (right arrow) next to the document name. Open the 'Action Menu' and select 'Email Participants'.

### Modifying Workflows

There are several actions you can take on a workflow once it has started. These actions are accessible vie the same methods used to email participants.

- Replace Participants: This allows you to chose a participant in the current workflow and another user to replace them with.
- Update workflow dates: This will allow you to change dates associates with a workflow, changing the due date for example.
- Cancel workflow: This will cancel the current workflow and remove tasks for users.

### Check Out, Check In and Undo Check Out

When it's time to make changes to documents, it's important to ensure that others cannot edit at the same time as you. Vault PromoMats uses check out and check in to acocmplish this. These options are available via the Action Menu.

- Check Out: To check out, open the 'Action Menu' and select 'Check Out'. The document will be locked for editing and the source file will be downloaded to your computer automatically.

- Check In: When edits are complete, you'll need to check the document back in. To check in, open the 'Action Menu' and select 'Check In'. You'll be prompted to choose the new version of the file from your computer and provide a description of the new version/changes (optional). The new file will be uploaded, a new viewable rendition will be generated, and the document will be available for other to check out.

- Cancel Check Out: If you check out a document and determine changes are not needed, you can cancel the checkout allowing others to work on the document. To cancel the check out, open the 'Action Menu' and select 'Cancel Check Out'.

### Uploading a New Document Version

Uploading a new document version will happen when a document has completed a review cycle or if you need to upload a second version of the document.

The check out and check in process we just covered is generally used before the review process begins during content creation.

- Open the 'Action Menu' and Select 'Upload New Version'.
- Click 'Choose' and select the new file from your computer.
- Click 'OK'.

The new version will be uploaded, a new minor version will be created, and a new viewable rendition will be rendered.

## Workin with Annotations

### Reply to an Annotation

During the review and approval process there will be time that you need to reply to an annotation. Most commonly, replies are used to ask questions of others involved or confirm that a request has been completed.

Working with annotations was made easier with Action UI.

- Enter into annotation mode by click the '+' icon.
- Select the annotation.
- Click into the text box and type your replay and click Reply.

When working with annotations you can mention another user by using the @ sign, a user selection dialogue will display and the user will be notified.

### Resolving an Annotation

When working through annotations, the ability to resolve an annotation helps to keep track of them.

- Enter into annotation mode bu clicking the '+' icon.
- Select the annotation.
- Click the checkmark in the header of the annotation.

The checkmark changes to a flag icon indicating it has been resolved.

### Reopening an Annotation

If you resolved an annotation by mistake or noticed an error after the fact, you are able to reopen it.

- Enter into annotation mode bu clicking the '+' icon.
- Select the annotation.
- Click the flag icon in the header of the annotation.

The flag icon will change to checkmark indicating the annotation is no longer resolved.

### Hiding an Annotation

As you progress through annotations you may choose to hide them. This method of hiding an annotation is different than in the annotation filters discussed later in this course.

- Enter into annotation mode bu clicking the '+' icon.
- Select the annotation.
- Click the ellipses in the header of the annotation.
- Select 'Hide'.

When you use this method to hide annotations, you must refresh the page to have them display again.

### Deleting an Annotation

You will only have tha ability to delete annotations that you have created.

- Enter into annotation mode bu clicking the '+' icon.
- Select the annotation.
- Click the ellipses in the header of the annotation.
- Select 'Delete'.

### Moving an Annotation

Annotations are attached to a specific location on a document. You can move an annotation by click the ellipses in the header of an annotation and selecting move.

- Note or Link Annotations with Text Selections: Moving the annotation will change which text is selected, you can also move from a text selection to an image selection.

- Note or link Annotations with Image Selections: Mobing the annotation will allow you to move or resize the selection, you can also move from an image selection to a text selection.

- Sticky Note Annotations: Moving the annotation will move the sticky note icon, while you can't select text, you can move it over a new text area. You cannot resize the sticky note icon.

- Line Annotations: Moving the annotation can move a line groupe to a different location on a document. You can also add, remove, or edit the lines in a groupe.

- Anchor Annotations: You can move any anchors that do not have inbound references, this requires the manage anchors permission.

When moving any annotation click the 'checkmark' icon to save it or the X to cancel the move.

### Using Annotation Filter

Action UI made it easier working with annotation filters. Now, when you select the filter menu, it stays on screen so you can easily change filters and see where you're filtered on.

- **Author:** Show annotations created by a specific user.
- **Note Status"** Show only open or resolved annotations.
- **Placement:** Show annotations that have a text or image selection or page-level.
- **Type:** Filter annotaions by the type of annotation.
- **Filter Anchors:** Show only anchors that are used by specific documents. A dialogue box will be displayed allowing you to search and select documents.
- **Show/Hide Annotations:** At the top of the filter menu you can switch between displaying or hiding all annotaions on the document.

### Determine Where an Anchor is Used

There may be instances where you need to see where a specific anchor in a reference document is used.

- Navigate to the reference document where the anchor exists.
- Click the '+' icon.
- Scroll to find the anchor of interest.
- Click on the Anchor Annotation.

Below the Anchor Name, a list of documents that reference the anchor will be displayed. You can expand each document to see details as to where the annotation exists on that document and link directly to it an pop-up window.

### Determine Where a Document is Used

Similar to being able to find where an anchor is used, you can do the same for a reference document for where a document link was created.

- Navigate to the reference document.
- Open the 'Action Menu'.
- Scroll and select 'Where Used'.

A report will display listing all documents that use the reference document. The name, relationship type, status, product, document type, and who it was created by will be displayed. From the report page, you can edit the columns displayed as well as export as a CSV or Excel document. All of these actions can be found in the Action Menu.

### Bring Forward Annotations

When working with new versions of a document, Vault PromoMats has a feature called 'Bring Forward Annotaions' that helps to save you time.

- Click the '+' icon.
- Click the right arrow icon.
- Select the version you want to bring forward from as well as the type of annotations you want to bring forward.
- Click 'Continue'.

The annotations will be brought forward and the page will refresh. If pages have been reorganized and/or text has been deleted, text selections will be brought forward as page level annotaions.

Image annotations will also be brought forward as page level annotations. You can use the move option discussed earlier in this course to move the annotations or in some cases you may delete the annotation.

### Exporting Annotations

Exporting Annotations on a document can be a time saver when comments are received in a review cycle. Vault PromoMats provides two ways to download a PDF that includes annotations.

1. PDF with Annotations Rendition

    This option will download a PDF with annotations in the margins.

    - To retrieve this PDF, click the down-arrow on the renditions icon next to the 'Action Menu'.
    - Select 'PDF with Annotations'.

2. Export Annotations

    This option will download a PDF with native comments that you can interact with. This option requires that your customer has offline annotations enabled.

    - To retrieve this PDF, open the 'Action Menu'.
    - Select 'Export Annotations'.

## Downloading Documents

### Downloading Renditions

Depending on the document type, there will be a number of renditions for a specific document. At a minimum, there will be a source document and a viewable rendition.

1. One way to dowload document renditions is located at the top of the document info page, you'll see an icon for the type of file the document is. You can click on that icon to download the source file or click on the down-arrow next to it to access other renditions.

    When performing a document search, you can access this same menu from the search results, saving you time.

2. The second way to access document renditions is via the document information panel.

    - Select the 'Document Files' section of the document information.
    - The viewable renditions and any other renditions that exist for the document will be displayed here.

### Bulk Download

From the document library and document viewer you can add documents to your cart to bulk download them. Click on the 'shopping cart' icon for the documents you want to add to your cart. In tabular view, this is not available.

Once you've added documents to your cart, the 'shopping cart' icon, next to the notifications icon, will display the number of items in your cart, to download, click the shopping cart. Click the 'Download' button to download all the files in one ZIP file or 'Empty Cart' to clear your cart.
