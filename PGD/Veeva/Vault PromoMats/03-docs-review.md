# Preparing Documents for Review

## Annotation Basics

Vault PromoMats includes robust annotation capabilities which allow you to add and reply to annotations as part of the review and approval process. An annotation consis of two parts:

- The selection of the document.
- The note, including any comments you made.

The ability to annotate documents is based on user permissions and the lifecycle state of documents. When it is available, entering into annotation mode is done by clicking the icon above the document viewer.

There are 6 types of annotations available in Vault PromoMats:

- Image.
- Text.
- Sticky Note.
- Link.
- Anchor.
- Line.

Switching between cursor modes to create the different types of annotation is easily done using the new selector drop-down located above the document viewer.

### Image, Text and Sticky Note Annotation

When selected, the cursor will change between text and image selection automatically when it recognizes selectable text.

- Text selections: Up to 200 words is supported.
- Image selections: Selection is limited to the boundaries of a single page.

Once your selection is made, you can add any comments and then click Save. For text annotations, you can also choose the type of comment you're making: highlight, insert, strikethrough, revise text, or external link.

### Line Annotation

Line annotations allow you to cross out or draw attention to content.

To create line annotation, click and drag to create the line. Click and dragging again will add additional lines to the group. Once you've drawn as many lines as you want, you can make any comments and choose the thickness of you line before clicking 'Save'.

To remove the lines you drew, click 'Cancel'.

### Link Annotation

Link annotations are used to create a link between two documents in Vault PromoMats. For review and approval this is used to create a link to supporting references for claims and other data in a promotional piece.

With the link annotation tool selected, drag and drop to select an image or use the cursor tool to select specific text. Once you make a selection a pop-up will appear for you to select an existing anchor, an entire document, or create a new anchor. One the anchor(s) and/or document(s) are select, click 'Close'.

Anchors are a specific selection of content within a document that are used when linking to supporting documents from a promotional piece. Anchors can be created ahead of time in a reference or supporting document or can be created as part of the process of creating a link annotation.

To create anchors, you must have the necessary document permissions.

To create an anchor on a document, select the anchor tool and select the relevant content. Once you've made your selection, you'll be prompted to name the anchor. Use a name that will make sense to others and follows any specific customer guidance.

## Working with Document Links and Anchors

### Creating Links on a Document

At times, you may need to Create a Link to a reference document or specific Anchor on a document that is no tied to a specific content selection.

- Go to the 'Relationships' section of the Document Information panel.
- Find the 'Linked Documents' panel.
- Click the '+' symbol.
- Search for the relevant document that you are linking to or where the anchors exist.
- Click the '+' symbol next to the relevant document(s) and/or anchor(s).
- Click 'Close'.

### Creating a Link to Content from Linked Documents

When referencing content, you can use existing documents and anchor links to make links to a selection of new content. This cuts down on the number of clicks required to create the link.

- Go to the 'Relationships' section of the Document Information panel.
- Find the Linked Documents panel and Expand it.
- Next to the document links or anchors that exist, click the triple dots icon.
- Select 'Link to Content'.
- Scroll to the relevant content in your document and select as you would when creating an annotation.
- Click the check to complete the new annotation.

### Importing Document Links

At times, content you create will use the same documents when referencing and annotating. To help with this process, Vault PromoMats allows you to Import Document Links from another document.

- Click the 'Action Menu'.
- Select 'Import Document Links'.
- Click the '+' symbol next to the document you wish to import from.
- Click 'Continue'.

These documents will now be in the Linked Documets panel where you can use the actions we previously reviewed to create a link to content.

## Working with Binders

Binders are a tool in Vault PromoMats that allows you to organize and group documents together into a structure that makes sense for your project.

Binder consis of sections, which can be nested to create a hierarchy and links to documents. The video gives an overview of binders, the rest of this section covers working with binders.

### Creating Binders

To create a binder, you must have the Create Binder Permission.

- Click the 'Create' button in the library.
- Choose 'Binder' and click 'Continue'.
- Select the document type that is relevant for the binder that you're creating.
- Click 'Next'.
- Complete the metadata for the binder as you would a document ensuring you're aligning to the customer specific process.
- Click 'Save'.

### Creating Binder Sections

Sections allow you to organize documents within a binder. Sections can be nested to create a hierarchy. A number will be shown next to each section noting the number of documents in it.

When navigating within a binder, the breadcrumb trail will allow you to navigate easily between your binder hierarchy.

- While in the binder, click 'Edit Binder'.
- Click 'Add'.
- Select 'Section' and name the section.
- Continue to add your sections, when done, click 'Done Editing'.

### Adding Documents to a Binder

Documents can be added directly to the binder or to a section within a binder. This creates a link to the document, it does not create a duplicate version of the document.

Any updates made to the document in the library or the state of the document will be reflected in the binder. To add the document(s):

- While in the binder, click 'Edit Binder'.
- By default, the document will be added to the root of the binder unless you click into a section.
- Click 'Add', select 'Existing Documents'.
- The document selector will open allowing you to search or use filters to find the documents you're looking for.
- Close the document selector and click 'Done editing' when you've added all documents.

### Uploading Directly into a Binder

In the instance you are adding a document that doesn't already exist in the library, you can do so from within the binder. You can either upload a new document or create a placeholder.

- While in the binder, click 'Edit Binder'.
- Click 'Add', select 'Placeholder' or 'Upload'.
- The process for doing either of these is the same as if you were adding to the library.
- Once the placeholder has been created or the document has been uploaded, click 'Done Editing'.

The document or placeholder created will be available in the library as well as the binder.

### Viewing and Filtering Binder Content

There are number of tools available to help you when navigating through binder content.

The document filter within a binder allows you to filter the content that is shown.

You can switch between compact view and outline view for a binder. With outline view selected, you easily expand the sections and view the content within them without having to click into each section. You can also take actions on the documents from this view via the 'Action Menu'.

### Reordering and Deleting Binder Content

When you're in edit mode on the binder, you drag sections to reorder them and even move documents between sections when in outline view.

If you need to remove content from a binder:

- First click 'Edit Binder'.
- Navigate to the section of the binder where the content exists.
- Click the 'Action Button' and Select 'Remove from Binder'.
- Click 'Done Editing'.

This only removes the content from the binder and will leave the content intact in the library.
