# Vault PromoMats Basics

## Vault Home

All Tasks is the default view which shows your tasks assigned to you and available tasks.

- From all the task views, you can click the document name to view the document or click 'Show more' to see details on the task.
- You can complete workflow and user tasks from these views by clicking 'Complete'.

My Tasks will display all tasks assigned to you and those you are responsible for completing. The tasks listed here may be steps in a workflow for a document or user tasks.

The Available Tasks list displays tasks that are available to you to accept and complete. These are tasks that are specific to your configured role in Vault PromoMats. One you accept a task from this list, it moves to My Tasks.

The Active Workflows view list all workflows which you are a participant in or are the owner of. Each workflow displays a link to the document, list of participants, their tasks, and the status of each.

## Library

The All Library view will display all the documents that your account has visibility to. Agency user's document access is generally limited to products your agency works on. You'll also have visibility to documents that you're directly involved in along with the supporting reference documents for the product(s) you work on.

The Recent Library view shows the last 20 documents that you viewed or worked on.

The My Library view will only show documents that you are the owner of or, those that you have checked out.

## Detail View

Detail View of content is available in bothe the document library and task lists. The layout can be changed by using the icon at the top of the library.

For tasks, detail view will display a link to the document as well as important task details. From this view you can expand to show additional details and complete the task.

For documents, detail view will display a thumbnail of the document, importnat document fields, and the state of the document. From this view, you can hover over the document to download the file or access the Action Menu.

## Library - Additional Views

Thumbnail view is available in the document library. Tile view organizes documents into a grid. For each document, the status, a thumbnail, the document name, and important document fields are displayed. In this view, the options to  download the content and to access the 'Action Menu' are always availbale without having to hover over the document.

Compact view is available in the document library. Compact view displays documents in narrow rows. The document thumbnail, name, status and other important fields are displayed. In this view, the options to download the content and to access the 'Action Menu' are available by hovering over the document.

Grid View of content is available in both the library and task lists. This view functions like a spreadsheet, you can resize and rearrange columns, add columns, sort by column values, and export document metadata from this view.

## Viewing a Document

Action UI has made it easier when viewing and working with a document in PromoMats. The Standrad view when working with documents is the same as Mixed View in the previous UI, where you see both the document and the related metadata.

The big difference in Action UI is that you can now resize the document viewer and document info panel by clicking and draggin the yellow line.

### Document Viewing Tools

Action UI has introduced a number of helpful tools for when you're viewing a document in PromoMats. These tools are found at the top-right of the document viewer window.

- **Rotate:** This allows you to rotate the current page you are viewing of a document clockwise. Dropdown icon allows you to rotate all pages in the document.

- **Zoom:** These allow you to zoom in or out of a document.

- **Document Fit:** This allows you enter full screen mode.

- **Fit Height/Width:** This allows you to change between fitting the document by width or height.

- **View Options:** This allows you to switch between the different viewing options:
  - Document View - the standard view.
  - Notes View - Displays any notes associated with the document.
  - Thumbnail View - Displays all pages in thumbnail format.

### Document Information Panel

Action UI brought changes to the document info panel; information has now been organized into different sections versus one long scrolling set of panels.

- **General Information:** General Information Tab will display all the panels with the standard/general information for the document.

- **Relationships:** Relationships Tab will display panels for relationships to the document your viewing, including Auto-Linked files, Components, Linked Documents, PI/SMPC, and other Related Documents.

- **Files:** Files Tab will display any of the related files to the documents and allow you to download them. Versions of files and ability to download are restricted by account permissions.

- **Sharing Settings:** Sharing Settings are essentially the permissions of each document. They define the user's role and permissions (allowed actions) on a document. From this panel you can:
  - Hover over a group name to view users that are part of the group.
  - Hover over a role to view permissions on the document.
  - If available, edit the Sharing Settings by clicking the Add button.
  - Sharing Settings are also accessible via the Action Menu.

  Sharing Settigns are also accessible via the Action Menu. Documents can only have on owner. If you add  another user to the owner role without adding yourself to a different role, you will potentially lose access to the document.

- **Timeline view:** Timeline View allows you to easily view the history of the document, including workflows and lifecycle state changes. From the Timeline view you can:
  - View a previous version by clicking on the version number.
  - View the Workflow History Report.
  - See the date(s) on which the state changed.
  - Use the 'Action Menu' to modify workflows.

## Searching Content

### Simple Search

The search box at the top of the interface allows you to easily search for documents.

- Click in the search box.
- Enter your search term, this might be the document name or author, or it could be other metadata information.
- Predictive results and search terms will be displayed as you type.
- You can click on a document in the predictive search, or click the magnifying glass or hit Enter/Return to display the search results and open the document from there.

### Advanced Search

Advanced search allows you to perform more complex queries when looking for documents.

- Fill an as much search cirteria as needed.
- Including content in your search will seach beyond the metadata into the document intself.
- Using all of these words or any of these words is similar to using the AND or OR function when searching in other platforms.
- Click search to display the results.

Only documents that you have permission to view/access will be returned in search results.

### Using Filters

Filters allow you to filter tasks, documents, etc. based on the related metadata. Use of filters is like filtering when using an e-commerce website.

- Click on the filter heading to expand and show relevant options.
- Click the value that you're searching for, the example in the left illustrates filtering for documents that are Approved for Use.
- If the filter you want to use is not displayed, click the pencil icon which will allow you to choose document fields to use as filters.

## Common Tools

### Using Favorites

Vault PromoMats allows you to mark documents as favorites, allowing you to quickly access these documents along with configuring notifications for them.

- Click the star to the left of the document in the library, these will display in detail or compact view.
- To access your favorite documents, click 'Favorites' in the view options of your document library.

### Action Menu

One of the big changes in Action UI was the replacement of the icon for the Action Menu, now represented by ellipses. The Action Menu is used throughout the Vault PromoMats interface and is the menu that will allow you to select various actions relevant to where you are in the interface.

Some examples of actions available vie the menu are:

- Upload new version.
- Check a document out/in.
- Create a draft.
- Export Annotations.
- Perform Bulk Actions.

Actions available via the Action Menu will vary by:

- Customer configuration.
- Acciont permissions.
- Workflow stage of document or object.
- Point of access (Library, Task list, etc.)

### Workflow Actions Menu

In the legacy UI, workflows were accessed via the Action Menu. In the Action UI, there is now a separate button to initiate workflows.

If you have permissions to start a workflow, you'll see the icon for the Workflow Actions Menu. When you click that, you'll see the icon for the Workflow Actions Menu. When you click that, you'll see the workflow(s) that you have permissions to.

### Notifications

Vault notifications were previously only visible from the Vault home/tasks screen. In Action UI, they are now available anywhere in the interface. The notification icon is located to the left of the user profile icon and when the orange dot is displayed, it means that there are one or more new notifications.

To view you notifications, you can click on this icon and notifications will be displayed in a hover menu, allowing you to remain where you were in Vailt. You can choose to click on the document the notification is realted to and quickly jump to that document.

## Creating a Placeholder

Placeholders allow you to create a document in Vault before the content is ready. This allows for metadata to be populated, a document/job number to be created, and workflows to be initiated.

1. Click the blue 'Create' button.
2. Select 'Placeholder'.
3. Click 'Continue'.

As with the creation of any document, you will need to classify the document. Document types and subtypes will vary from customer to customer, it's important to understand each customer's process and taxonomy.

- You can click into the document type field and scroll through available types.
- You can begin to type the document type and select from the predictive list.
- You can click the binoculars to select the type, subtype and classification (if applicable).
- Once you've found the document type, click 'Next'.

Next step is to populate the metadata for the document. At a minimum, the required fields which will be highlighted will need to be completed. Taxonomy and required fields will vary between customers.

- Fill in any text fields.
- Select the appropriate values for any radio button or drop-down fields.
- It is important at this step to understand your customer's process to ensure correct population of the metadata.
- When metadata is complete, click 'Save'.

The placeholder is complete once the metadata is saved. As you can see to the right, there is a document number associated with this document that can be included during content creating.

Once the document is ready, uploading to the placeholder is easily done by clicking the Upload File button and selecting the document from your computer.

## Creating a Document

When placeholders are not used, the initial creation of the document will be done along with uploading the content to Vault PromoMats.

1. Click the blue 'Create' button.
2. Select 'Upload'.
3. Click 'Continue'.

From here you can either drag and drop the file you are uploading into Vault PromoMats or click the Choose button to select it from your computer.

- You can click into document type field and scroll through available options.
- You can begin to enter the document type and select from the pedictive list.
- You can click the binoculars to select the type, subtype and classification (if applicable).
- Once you've found the document type click 'Next'.

Next step is to populate the metadata for the document. At a minimum, the required fields which will be highlighted will need to be completed. Taxonomy and required fields will vary between customers.

- Fill in any text fields.
- Select the appropriate values for any radio button or drop-down fields.
- It is important at this step to understand your customer's process to ensure correct populateion of the metadata.
- When metadata is complete, click 'Save'.

The creation of the document is completed once the metadata is saved. Generation of the viewable rendition will begin; this may take longer for larger files.

- Files up to 4GB can be uploaded via the user interface, larger files will require use of Vault FTP.
- For audio and video files, compressed versions under 4GB should be uploaded for a viewable rendition to be created.

### Creating a Draft

During the review and approval process, you'll be uploading new versions of a document as revisions are made. This will usually be accomplished by using the Create Draft functionality accessible via the Action Menu.

1. Click the 'Action Menu'.
2. Select 'Create Draft'.
3. Either copy the current version or select to upload the new file.
4. If uploading, click 'Choose' and select the file and then click 'OK'.

A new minor version of the document has been created. The ability to create drafts is controlled by permissions and may only be available to you on documents in specific workflow states.
