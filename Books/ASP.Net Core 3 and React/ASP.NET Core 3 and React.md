ASP.NET Core is an open source and cross-platform web application framework built by Microsoft. Is a great choice for building highly performant backends that interact with databases such as SQL Server that are hosted in the cloud, such as in Microsoft's cloud platform, Azure.

React was built by Facebook in order to improve the scalability of their code base, and was eventually open sourced in 2013.
