## Reports and Dashboards

Reports and dashboards can be created using Engage for Portals multichannel data which can consolidate activities from multiples users and channels.

Veeva provides a set of standard Engage for Portals reports available in CRM online.

## Default Engage for Portals Reports

| Reports                                 | Description                                                                                                                                                                                                                                  |
| --------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Engage Slide Views by Product**       | This report displays the number and percentage of views for each product in the last 90 days. Filters added in the report - a) Event in Multichannel Activity Linea equals Slide View and b) Multichannel Activity record type equals Engage |
| **Engage Visits by Account Speciality** | This reports shows the number of visits grouped by Account specialty in the last 90 days. A filter is added in the report - Multichannel Activity record type equals Engage                                                                  |
| **Engage visits by Country**            | This reports displays the number of visits from each country in the last 90 days. A filter is added in the report - Multichannel Activity record type equals Engage.                                                                         |
| **Engage Visits by Device and Browser** | This reports displays the number of visits grouped by devices and further grouped by the browser in the last 90 days. A filter is added in the report - Multichannel Activity record type equals Engage.                                     |
| **Engage Visits by Site**               | This report displays the number of visits for each site in the last 90 days. A filter is added in the report - Multichannel Activity record type equals Engage.                                                                              |
| **Presentation Views by Month**         | This report displays the number of views for each presentation in each month for the last 90 days. A filter is added in the report. - Multichannel Activity record type equals Engage.                                                       |

## Reporting on HCP Interactions

Report on any attribute related to multichannel activities and multichannel activity lines:

**Interaction settings**
- Duration, start/end time.
- Device.
- Browser.
- IP addresses.
- Operating system.
- Location (country, city).

**Content**
- Key messages.
- Landing page use.
- Viewport resolution.
- Total pages viewed.
- Time spent watching video.

**Other technical details**
- e.g., Mozilla/5.0 (Macintosh; Intel Mac OS X 10_13_1) AppleWebkit/537.36 (KHTML, like Gecko) Chrome/66.0.3359.139 Safari/537.36

## Reporting on Approved Emails

Report on any attribute related to sent emails and email activities.
- Clicks
- Opens
- Dropped, Bounced
- Unsubscribe

If Engage for Portals [link has been Labeled](https://crmhelp.veeva.com/doc/Content/CRM_topics/Multichannel/ApprovedEmail/ManageCreateContent/CreatingContent/LabelingLinks.htm), it can be used to filter and identify activities related to Engage for Portals distribution.
