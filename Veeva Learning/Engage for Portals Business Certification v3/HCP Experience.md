## Accessing the Presentation

HCP receives an email from rep or MSL and clicks on the link to be directed to the Engage for Portals content.

### Process

When the HCP clicks on the link, they will be taken to the specific website where they can view and interact with the Approved content from Vault which is displayed in the Engage Viewer.

- The most recently approved version of the content will automatically be displayed.
- If content has been removed from Vault, no content will be displayed.

## Using the Engage Media Player Interface

### Basic Controls

The Engage media player includes basic controls that allow users to navigate through the content, modify player size and view slide thumbnails. The media player controls are non-configurable. [^1]

[^1]: Swipe natively works for images and video slides. On HTML slides, the content creator must code in the recognition of the swipe gesture.

### Select a Slide

Thumbnails are rendered at the size they were when uploaded into Vault.

- Use the mouse or keyboard arrow keys + space bar to select a slide.
- The 'Exit Thumbnails' arrow button or the ESC key will return the user back to the slide previously viewed before bringing up the thumbnails screen.

### Additional Controls

- The up and down arrow buttons allows users to move through pages of a PDF. You are also type the page number to jump to a particular page.
- Video slides has a pause/play button and a volume control button. Users can also use the red scrubber bar to move quickly through the video.
