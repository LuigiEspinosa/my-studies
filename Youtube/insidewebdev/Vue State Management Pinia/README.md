# Vue State Management with Pinia - Pinia Crash Course by insidewebdev

## Pinia vs Vuex

- Composition API.

- Better Developer Experience

  - No more magic string to inject.
  - Import the functions, call them, enjoy autompletion!

- Amazing Typescript Support

  - No need to create custom complex wrappers to support Typescript.
  - Everything is typed and the API is designed in a way to leverage TS type inference as much as possible.
