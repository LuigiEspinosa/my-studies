## Introduction to Chat GPT and Prompt Engineering

### What is a Prompt?

A prompt is a statement or question that you send to Chat GPT to start a conversation.

### How Do Prompts Work in Chat GPT

Prompts work by providing a starting point for the AI language model to generate a response. Chat GPT uses its language processing algorithms to analyze the prompt and responds to it.
### Components of a Prompt

- Instruction: Directive for Chat GPT on action to perform.
	- Simple: One direct instruction in a prompt.
	- Complex: Two or more instructions in a prompt.

- Primary Content: Text to transform or process as per instructions.
	- Simple: A short sentence to be processed.
	- Structured: Contains a pattern in provided text.
	- Unstructured: No pattern exists in provided text.

- Example: Sample response to generate similar ones.
	- Zero Shot: Prompt contains no example of the desired result.
	- One Shot: Prompt contains one example of the desired result.
	- Few Shot: Prompt contains two or more examples of the desired result.

- Cue: Any text or symbol to be used as a prefix in a response.
	- Zero Cue: No cue is given to Chat GPT.
	- One Cue: One cue is given to Chat GPT.
	- Multiple Cue: Multiple cues are given to Chat GPT.

- Supporting Content: Contextual information to influence the response.

### Types of Prompts

 - Chained: A series of prompts.
 - Conditional: A series of conditions.
 - Open Ended: No restrictions.
 - Structured: A well-organized prompt.

### Action Verb to Create Prompts

Action verbs are words that describe an action or a process and the tell Chat GPT exactly what you want it to do.

- Analyze, Define, Outline, Suggest, Arrange, Create, Explain, Rephrase, Clarify, Differentiate, List, Rewrite, Combine, Discuss, Recommend, Summarize.

### How to Write a Meaningful Prompt?

First, make sure your prompt is clear and concise by providing specific information to ensure that Chat GPT understands what exactly you want it to answer.

Second, ensure that prompt is descriptive and includes as much detail as possible to allow Chat GPT to understand the context property. 

Third, you should ensure that the order or chronology of tasks that you give to Chat GPT is as per your intended outcome.

Finally, ensure that there is no bias in the prompt. Try to avoid asking leading questions or questions that are based in any way.

## Mastering Prompt Crafting

### Introduction to Basic Prompt Crafting Techniques

- Use Delimiters:
	- Special characters and symbols.
	- Used to separate different parts of input to text.
	- Tells Chat GPT to treat each section differently.

- Include Details:
	- Provides clear instructions or context.
	- Enhances accuracy by offering specific information.
	- Encourage completeness.

- Using Markdowns to Enhance Prompt Clarity.

### Advanced Prompting Strategies

- Direct Instructions:
	- Provides clear and specific information.
	- Guides the AI model to tailor response as desired.
	- Useful when user has a precise goal or topic in mind.

- Specific Contextualization:
	- Constrains Chat GPT to respond within a particular context or domain.
	- Guides Chat GPT to respond directly to a specific topic, ensuring depth and accuracy.

- Creative Scenario Setting:
	- Encourages Chat GPT to envision imaginative scenarios.
	- Directs AI to engage in creative thinking.

- Role Play and Simulation:
- Comparative Analysis.
- Progressive Inquiry.
- Problem-Solving Framework.
